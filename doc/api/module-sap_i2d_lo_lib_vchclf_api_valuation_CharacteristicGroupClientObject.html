Module: sap/i2d/lo/lib/vchclf/api/valuation/CharacteristicGroupClientObject

Implementation of the Characteristic Group Client API. Each VC UI Extension view controller (which needs to be derived from module:sap/i2d/lo/lib/vchclf/api/valuation/controller/CustomCharacteristicGroupBase ) owns an instance of this object and has access. It allows to retrieve Characteristic data independent of the context to which a module:sap/i2d/lo/lib/vchclf/api/valuation/control/CustomCharacteristic is bound. Furthermore loading of Domain Values can be manually triggered (e.g. for an custom Value Help implementation) and value assignments can be made. This is especially helpful when it comes to own event handler implementations if the provided ones are not sufficient.
