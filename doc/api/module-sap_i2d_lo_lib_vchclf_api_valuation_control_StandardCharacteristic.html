Module: sap/i2d/lo/lib/vchclf/api/valuation/control/StandardCharacteristic

Represents a Standard Characteristic within a VC UI Extension view. This means it leverages the complete infrastructure and behaves in the same way as in a 'standard' Characteristic Group except that it can be freely placed. Therefore event handling and all other relevant bindings are automatically managed. The inherited aggregation content serves the stanard Characteristic control and must therefore not be used. Note that for each Characteristic whithin a Characteristic Group there can be only one occurence of a Standard Characteristic. This limitation does not exist for Custom Characteristic controls module:sap/i2d/lo/lib/vchclf/api/valuation/control/CustomCharacteristic.
