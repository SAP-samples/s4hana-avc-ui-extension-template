Module: sap/i2d/lo/lib/vchclf/api/valuation/control/CharacteristicContextContainer

Base class for controls bound to a Characteristic Context specified by its name. Child elements need to be placed in the default aggregation content. They automatically inherit a binding context with the name view which is actually bound to the Characteristic entity specified with property name. This class must not be used directly. See the sub-classes: module:sap/i2d/lo/lib/vchclf/api/valuation/control/CustomCharacteristic and module:sap/i2d/lo/lib/vchclf/api/valuation/control/StandardCharacteristic
