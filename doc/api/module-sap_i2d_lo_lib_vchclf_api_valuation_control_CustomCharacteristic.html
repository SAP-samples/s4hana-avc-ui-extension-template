Module: sap/i2d/lo/lib/vchclf/api/valuation/control/CustomCharacteristic

Represents a Custom Characteristic within a VC UI Extension view. The custom visualization needs to be placed in aggregation content and can make use of available UI5 or own controls. Each instance has a CharacteristicValueProcessor attached that becomes involved as soon as a value assignment should be performed. This processor is decoupled from the actual control representation and implictly used when a generic event handler of module:sap/i2d/lo/lib/vchclf/api/valuation/controller/CustomCharacteristicGroupBase is called.
